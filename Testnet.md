###Запуск Testnet
Тестовая сеть-это ваша Безопасная среда, в которой устойчивость вашей валюты и сети может быть проверена до запуска или до основного обновления. Как правило, вы не хотите разворачивать функцию, которая может поставить под угрозу пользовательский опыт или работоспособность сети.

Прежде всего вам следует запустить Daemon с аргументами:
```
--testnet и --data-dir
```
Это позволит принудить Daemon к запуску тестовой сети, а также создать новую папку для сохранения данных созданной сети.

Например:

```
testcoined --testnet --data-dir=new / path/to / blockchain
```
###Подключение кошелька
Подключение реализует очень просто. Достаточно указания аргумента 
```
--testnet
```
Образец:
```
simplewallet --testnet
```
###Подключение к тестовой сети дополнительных узлов
Для работы Testnet необходимо и достаточно наличия одного узла. Тем не менее, имеется возможность подключить дополнительные узлы:
```
cryptonotecoind --testnet --data-dir=new / path/to / blockchain --add-exclusive-node=testnet.node.IP.port
```
